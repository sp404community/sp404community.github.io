## Горячие клавиши сэмплера Roland SP-404MK2 с прошивкой версии `v.5.0.1`

Сочетания с кнопками: [SHIFT](#shift), [REMAIN](#remain), [VALUE](#value), [DEL](#del), [COPY](#copy), [MFX](#mfx), [DJ mode](#dj-mode), [TR-REC](#tr-rec)

## SHIFT
Удерживая нажатой кнопку [SHIFT]
```
Пэд [1]
    FIXED VELOCITY
    Даже слабое нажатие на пэд будет звучать, будто максимально сильное.

Пэд [2]
    16 VELOCITY
    Выбор 16 ступеней силы нажатия (громкости) пэда.

Пэд [3]
    CUE
    Регулировка баланса звука, который вы слышите через выход на наушники.

Пэд [4]
    CHROMATIC
    16 вариантов питча выбранного пэда.

Пэд [5]
    EXCHANGE
    Поменять местами два пэда - в режиме сэмплов или паттернов.

Пэд [6]
    INIT PARAM
    Сбросить параметры выбранного пэда на дефолтные (Pitch, Speed и пр.)

Пэд [7]
    PAD LINK
    Позволяет связать несколько пэдов, чтобы нажатие на любой запускало все одновремено.

Пэд [8]
    MUTE GROUP
    Группирует пэды так, чтобы одновременно мог играть только один из группы. Следующий выключит предыдущий.

Пэд [9]
    METRONOME
    Включение/выключение метронома.

Пэд [10]
    COUNT-IN
    Добавляет отсчет перед началом сэмплирования или записи паттерна.

Пэд [11]
    TAP TEMPO
    Позволяет задать темп интуитивно понятным способом, постукивая по пэду в такт, как будто хлопаете под музыку.

Пэд [12]
    GAIN
    Отображает вкладку UTILITY MENU > SYSTEM > GAIN.

Пэд [13]
    UTILITY
    Отображает экран UTILITY MENU.

Пэд [14]
    IMPORT/EXPORT
    Отображает UTILITY MENU > IMPORT (меню импорта / экспорта).

Пэд [15]
    PAD SETTING
    Отображается меню UTILITY MENU > PAD SET (настройка пэдов).

Пэд [16]
    EFX SETTING
    Отображается экран UTILITY MENU > EFX SET (настройка эффектов).

[BUS FX]
    MUTE BUS
    Временно отключает звук, передаваемый на шину (звук воспроизведения сэмпла или звук, поданный на гнездо INPUT), и выводит только звук эффекта.

[HOLD]
    PAUSE
    Приостанавливает воспроизведение сэмпла, который воспроизводится в данный момент.

[EXT SOURCE]
    INPUT SETTING
    Отображает экран настроек входных сигналов.

[SUB PAD]
    PROJECT
    Отображается экран выбора проекта.

[MARK] (зажать на три секунды)
    SAVE EFX PARAMETER
    Сохраняет основные параметры эффектов, назначенных на BUS 1 и BUS 2.

[PITCH/SPEED]
    ENVELOPE
    Устанавливает, как изменяется во времени громкость сэмпла.

[START/END]
    CHOP
    Нарезает сэмпл по позициям маркеров и назначает полученные сэмплы на новые пэды.

[PATTERN SELECT]
    UNDO
    Отменяет только что введенные (записанные) данные. Это дейстует только во время записи паттерна

[ROLL]
    ROLL SET
    Покажет, с какой частотой будет повторяться сэмпл в режиме ROLL (барабанной дроби).

[REVERSE] + пэды [1]-[16]
    Pad MUTE
    Включение/выключение звука выбранных пэдов.

[REVERSE] + [REMAIN]
    Pad MUTE MODE
    Задержит кнопки [SHIFT] и [REVERSE] в нажатом состоянии, чтобы удобнее мьютить и размьючивать пэды.

[LOOP]
    PING-PONG LOOP
    Зацикливание сэмпла вперед-назад.

[GATE]
    GATE ALL ON/OFF
    Включает/выключает параметр GATE для всех сэмплов в выбранном банке.

[BPM SYNC]
    SYNC ALL ON/OFF
    Включение/выключение параметра BPM SYNC для всех сэмплов в выбранном банке.

[REC]
    TR-REC
    Переключение на запись паттерна с помощью TR-REC. Это позволяет записать сэмпл на воспроизводимый паттерн. Действует только при воспроизведении паттерна

Кнопки банков [A/F]-[E/J]
    BANK VOLUME
    Регулирует громкость для указанного банка в целом.

[REMAIN] (удерживать три секунды)
    LIVE MODE
    Отключение кнопок, которые не используются при живом выступлении (кнопки сэмплирования и редактирования).

[COPY]
    BANK PROTECT
    Защищает сэмплы и паттерны, назначенных на пэд от копирования, случайной перезаписи при редактировании или удаления.

Ручка [VALUE] (нажать)
    MARK
    Переключение функции кнопки [MARK]: режимы SBS / Looper.

[RESAMPLE]
    SAMPLE MERGE MODE
    Режим объединения сэмплов.

[НАСТРОЙКА ЗАПИСИ]
    SOUND GENERATOR MODE
    Переключение в режим генератора звука (синтезатора).

[EXIT]
    STOP
    Остановка воспроизведения всех сэмплов.

```

## REMAIN
Удерживая нажатой кнопку [REMAIN]
```
Пэды [1]-[16]
    Выбор шины эффектов для пэда: Bus 1 /
    Bus 2 / мимо всех

Банки [A/F]-[E/J]
    Устанавливает шину эффектов для всех
    пэдов в банке.

[MFX]
    Задержит экран редактирования эффекта.

[BUS FX]
    Поменять местами эффекты BUS 1 и BUS 2.
    Значения параметров эффектов сохранятся.

Редактирование следующих параметров на экране Pitch/Speed:
 SPEED
 PITCH
 VOLUME
 PAN
 BPM
 BPM SYNC
 GATE
 LOOP
 REVERSE
    Одновременно редактировать параметры в
    одном банке.

Редактирование следующих параметров на экране Envelope:
 ATTACK
 HOLD
 RELEASE
 BPM SYNC
 GATE
 LOOP
 REVERSE
    Одновременно редактировать параметры в
    одном банке.

```

## VALUE
Удерживая нажатой ручку [VALUE]
```
Пэды [1]-[16]
    Выбор пэда без воспроизведения звука.
    Так же выбирается паттерн без включения,
    в режиме [PATTERN SELECT].

[BUS FX]
    Временно отключает эффекты на Bus 3 и
    Bus 4.

[SUB PAD]
    Это устраняет минимальные различия во
    времени при одновременном пуске
    нескольких сэмплов.

Кнопки эффектов [FILTER+DRIVE]-[MFX]
    Включает эффекты кратковременно - только
    при удержании их кнопок (EFFECT GRAB).

[GATE]
    Включение/выключение режима
    воспроизведения сэмплов "one-shot
    playback".

```
-----

## DEL
Удерживая нажатой кнопку [DEL]
```
[MARK]
    Удаляет из паттерна автоматизацию
    эффектов, записанную с помощью EFX
    MOTION REC. * Работает только на экране
    Sample Edit

[REVERSE]
    Удаляет из паттерна автоматизацию
    заглушения пэдов, записанную с помощью
    EFX MOTION REC. * Только на экране
    Sample Edit

[EXIT]
    Удаляет ВСЕ сэмплы или паттерны в банке.

```
-----

 VOLUME

 BPM
 BPM SYNC
 GATE
 LOOP
 REVERSE

Редактирование параметров Envelope:
 ATTACK
 HOLD
 RELEASE
 BPMSYNC

```

```

    Отображает список MFX LIST, начиная с

## DJ mode
Ярлыки, используемые в режиме DJ
```
[SHIFT] + [REVERSE] + [13] (CH1)
[SHIFT] + [REVERSE] + [15] (CH2)
    MUTE
    Отключает звук воспроизводимого сэмпла.

[SHIFT] + пэды [1]-[16]
    Воспроизводить сэмпл с позиции соотв. маркера, установленного для этого сэмпла.

[SHIFT] + [REMAIN]
    [SHIFT] остается в "нажатом" состоянии. Это облегчает выбор маркера и воспроизведение. Нажмите кнопку [EXIT], чтобы отменить это поведение.

[SHIFT] + [MARK]
    Добавлять маркеры во время воспроизведения сэмплов в режиме DJ.

[SHIFT] + [START/END]
    Редактировать маркеры во время воспроизведения сэмплов в режиме DJ.

[SHIFT] + [DEL] + пэды [1]-[16]
    Удалить маркеры для текущего сэмпла.

[SHIFT] + [ROLL]
    ROLL SIZE
    Устанавливает интервал дроби Roll: 1/4, 1/2, 1 или 2 такта. Установите ROLL SIZE перед воспроизведением дроби. (Во время воспроизведения дроби уже не получится изменить ROLL SIZE).

[ROLL] + [13] (CH1)
[ROLL] + [15] (CH2)
    Повторяет сэмпл с более частым интервалом. Если ROLL SIZE (интервал) больше, чем длина сэмпла, то воспроизведение ролла невозможно.

[ROLL] + пэды [1]-[4]
    Изменение интервала дроби во время его воспроизведения. [ROLL] + [1]: (1/4), [ROLL] + [2]: (1/2), [ROLL] + [3]: целая, [ROLL] + [4]: два такта

[REMAIN] + [14] (CH1)
[REMAIN] + [16] (CH2)
    BUS FX
    Установить шину, на которую будет отправлено воспроизведение сэмплов CH1/CH2.
Удерживая нажатой кнопку [REMAIN], при каждом нажатии на [14] или [16] переключается:
BUS-1 - BUS-2 - DRY - BUS-1.

[RESAMPLE] + нажать [VALUE].
    VOLUME CURVE
    Выбор характеристик кривой громкости, используемых для каждого регулятора (CH1 LEVEL, CH2 LEVEL, X-FADE).
При нажатом [RESAMPLE] каждое нажатие ручки [VALUE]:
FAST CUT - LINEAR - SQUARE - CUBIC - FAST CUT.

[DEL] + [2] + [6] (CH1) или [DEL] + [4] + [8] (CH2)
    BPM
    Сброс темпа на значение по умолчанию.

[START/END]
    Переключение между функциями CUE MIX или X-FADE для ручки [CTRL 3].

[PITCH/SPEED]
    Изменяет количество цифр для BPM. При каждом нажатии [PITCH/SPEED] дисплей переключается: целые числа - до второго десятичного знака - до первого десятичного знака - целые числа. При установке значения BPM с помощью пэдов [2] [4] (BPM+) и пэдов [6] [8] (BPM-), количество цифр изменяется в зависимости от минимальной единицы измерения.

[MARK]
    Переключение между экранами EFX и MIXER.

[BPM SYNC]
    Выбор канала (CH1/CH2) для воспроизведения в обратном направлении кнопкой [REVERSE].

```
-----

## TR-REC
Шорткаты в режиме TR-REC
```
[DEL] + банк [A/F]
    Удаление в текущем такте всех нот,
    соответствующих выбранному пэду.

[DEL] + банк [B/G]
    Удаляет в текущем такте вообще все ноты.

[ROLL] + ручка [CTRL 1]
    Запись движения ручки [CTRL 1].
    Только в MODE = TRIG.

[ROLL] + ручка [CTRL 2]
    Запись движения ручки [CTRL 2].
    Только в MODE = TRIG.

[ROLL] + ручка [CTRL 3]
    Запись движения ручки [CTRL 3].
    Только в MODE = TRIG.

Ручка [VALUE] (нажать) + [SUB PAD] + пэды [1]-[16]
    Выбирать сэмплы без их воспроизведения.

```
